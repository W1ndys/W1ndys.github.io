谨此纪念我的 Github 博客历史

用 Github Pages 搭建博客已经有一段时间了，一共写了 80 篇博客，其中有些是技术文章，有些是生活感悟，也有些是随笔。

也记录了选课指北和速通指南，起初这些只是一个普通的 README，后来开发了 2.0 版本，已经成为一个正式的文档站。

从最开始的看教程搭建博客，搞了三整天，到现在写过无数脚本来提高工作效率，也实现了源码+静态页面同时备份，实现多端同步，也有了很多收获，也有很多想法。

在这期间，我对 Github 也有了更深的了解，回望过去，2022 年 7 月 16 日，我第一次加入 Github,那时候还是个小白，啥也不懂，也不懂什么是源代码托管，到 23 年 7 月 3 日，我创建了我的第一个 repo，那时候还不是代码，我只是写了一个 README，用来收集常用的网站。2023 年 7 月 22 日我发出了我的第一个issues，，没想到那竟然是与安卓逆向相关的，我已经没有任何印象了。7 月 28 日，创建了算是自己的第一个代码仓库，是我写的用 C++演奏音乐，演奏的犁牛之子歌，当时还给校新媒体的师姐沟通，发到曲师大官方抖音了。2023 年 9 月 11 日，在 fluid 仓库里发了一个issues，那应该是我真正了解什么是 issues 之后的第一个。12 月 3 日，我开始尝试使用 ChatGPT 写脚本来提高我的日常工作效率，生成了第一个脚本，九宫格切图，并开源到了 Github。12 月 10 日，我在统计表格的时候突发奇想，既然我做的都是机械化的工作，能不能用脚本来提高效率，于是我上传了我第二个脚本，Excel 日期更新。

我 2023 年的 Github 生涯已经基本接近尾声了。

真正让我对 Github 了解程度突飞猛进的时间是 23 年的寒假。因为我本来就打算换博客主题了，但是没想到两个主题的依赖有冲突，这让我不得不删库，重新建立博客，于是我又重新开始了我的博客之旅。从安装 Hexo 到配置主题，到部署，再到域名绑定，到评论系统的切换，到网站的镜像备份，我花了大量的时间，部署完毕之后还有一大堆 bug 要修。就因为这个我整整一个月都在搞博客。

我 22 年 23 年加起来才 19 次，但 24 年到四月，就已经一千三百多次了。

24 年的 Github 贡献我就先不写了，等 25 年开始再写吧。

最开始的博客主题是 fluid，也就是这个页面的博客，我的新博客主题已经换成了 butterfly，感谢两个博客的作者。

从最开始的看教程魔改，到我自己用 js 魔改自己想要的功能，博客魔改也加深了我对前端 HTML、CSS、JavaScript 的理解。到后来接触 Netlify、Cloudflare、Vercel、Render、Zeabur 等静态网站托管平台，也让我对网站的部署和运维有了更深入的理解。

从刚开始做博客的时候，那时候天天搞博客美化，到了最后，博客回归了原貌，果然美化的尽头是原生

博客的评论系统也由 waline 换成了 Giscus，评论系统的功能更加强大，也更加方便用户评论。

我的博客站点已经转为了：https://blog.w1ndys.top/
